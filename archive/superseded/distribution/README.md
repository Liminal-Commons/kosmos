# Superseded: Distribution Models

These documents describe earlier approaches to oikos distribution that have been superseded by the **federation model** documented in KOSMOGONIA.md.

## What Changed

The original distribution model used static distribution channels and explicit push/pull operations. The federation model replaces this with:

- **Continuous sync** via syndesmos bonds
- **Same mechanism** for expressions, theoria, and oikoi
- **Time-sensitive execution** (expressions sync fast, oikoi sync deliberately)
