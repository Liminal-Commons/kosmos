# theoria/stoicheia-representation-layers

**Eidos:** theoria

## Data

- **domain:** dynamis
- **insight:** Stoicheia exist at three representation layers: arche-level (YAML definitions interpreted by host), entity-level (loaded as queryable entities), and WASM-level (portable modules). These aren't mutually exclusive — a manifest declares which form each stoicheion takes. Arche-level is current reality; entity-level enables tooling like pege emission; WASM-level enables portability. The progression is additive, not replacement.
